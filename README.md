# Projeto Game CES22

Título: Ultimate Runners

Python 3.9.1

Pygame 2.0.1

# Instruções de Instalação 

```
git clone https://github.com/JorgeHohhe/UltimateRunners.git
cd UltimateRunners
pip install -r requirements.txt
python3 MainGame.py
```
